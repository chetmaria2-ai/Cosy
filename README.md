# 🏠 CosyHome

> Десктопное приложение для онлайн-магазина товаров для дома — уютный шопинг прямо с рабочего стола.

---

## 📋 О проекте

**CosyHome** — это JavaFX-приложение с подключением к PostgreSQL, реализующее полноценный интернет-магазин товаров для дома. Поддерживает два режима работы: для зарегистрированных пользователей и для гостей.

---

## ✨ Функциональность

### 👤 Для пользователей
- Регистрация и авторизация
- Гостевой режим (просмотр каталога без входа)
- Каталог товаров с фильтрацией по категориям
- Корзина с управлением количеством товаров
- Личный профиль с бонусными баллами лояльности
- Отзывы и рейтинги товаров
- Служба поддержки

### 🛠️ Для администраторов
- Панель администратора
- Управление товарами, пользователями и отзывами
- Система разрешений (permissions)
- Журнал действий администраторов

## ⚙️ Установка и запуск

### Предварительные требования
- Java 17 или выше
- PostgreSQL
- Maven или IntelliJ IDEA

### 1. Клонируй репозиторий
```bash
git clone https://github.com/chetmaria2-ai/diyavol.git
cd diyavol
```

### 2. Настрой базу данных

Создай базу данных в PostgreSQL:
```sql
CREATE DATABASE prilo;
```

В файле `DatabaseConnection.java` при необходимости измени параметры подключения:
```java
private static final String URL = "jdbc:postgresql://localhost:5432/prilo";
private static final String USER = "postgres";
private static final String PASSWORD = "твой_пароль";
```

### 3. Запусти приложение

Через IntelliJ IDEA: открой проект и запусти `HelloApplication.java`

Через Maven:
```bash
mvn javafx:run
```
